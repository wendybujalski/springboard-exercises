Part One

  1. HTTP stands for Hypertext Transfer Protocol - it is a standard used for the transfer of data to and from servers.
  2. URL stands for Uniform Resource Locator - it is a human-readable address for a particular resource/website on the internet.
  3. DNS stands for Domain Name System - it takes human-readable URLs and converts them to the appropriate IP addresses.
  4. A query string is an extra string added to the end of a URL which adds additional query data to it in the form of key/value pairs.
  5. GET is used to request data from a server without modifying anything. POST is used to send some data to the server to save/use.
  6. An HTTP request is an individual request to a server from a client for something (HTML/CSS/etc) using the HTTP protocol.
  7. An HTTP response is the response sent to the client from the server after an HTTP request is made. It might include the requested resource, an error code, or additional information.
  8. An HTTP header is attached to every HTTP request/response and ccontains additional information about it. Examples include Host, Content-Type, Cache-Control, Accept, and more!
  9. The browser converts the URL into an IP address using the DNS, then makes an HTTP request to that IP address including relevant header information. It then waits for the server to respond - if the response is successful (status code 200) then the browser generates a DOM from the given HTML and sends requests for any additional resources needed for that page. Finally it displays the page.
